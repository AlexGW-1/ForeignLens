# ForeignLens
ForeignLens — платформа для мониторинга и публикации международной политической повестки через призму зарубежных источников. Мы собираем материалы “первой линии” из разных стран и языков, приводим их к единому формату, переводим, дедуплицируем и показываем несколько углов зрения на одно событие.
